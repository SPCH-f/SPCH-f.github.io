Advanced autonomous smart greenhouse ecosystem integrating Reinforcement Learning inference with a containerized time-series MLOps data pipeline. The infrastructure utilizes a decoupled client-server architecture communicating via TCP/IP socket programming, which was expanded to stream real-time operational states and Q-Learning action logs directly into a Docker-hosted MySQL production environment. To evaluate data analytics capabilities and storage scaling, a high-velocity script was deployed to generate 100,000 historical rows for bulk ingestion stress testing. Engineered analytical queries utilizing aggregation and conditional logging (SUM(CASE WHEN)) to extract agent behavioral profiles under severe climate variations. Conducted performance tuning and execution profile evaluation, discovering query runtime trade-offs regarding index lookup overhead on low-cardinality attributes. Outcome: Designed a production-grade data engineering simulation demonstrating end-to-end data ingestion pipelines, analytical report execution, and architectural database tuning.
