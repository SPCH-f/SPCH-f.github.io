End-to-end sports facility management application that digitalizes court and equipment reservations to completely replace legacy paper-based tracking methods. The system pairs a dynamic React.js frontend interface for real-time calendar and state synchronization with a scalable Node.js backend architecture. Security boundaries are enforced via strict Role-Based Access Control (RBAC), allowing automated validation of booking submissions and reducing manual admin overhead. Outcome: Delivered a streamlined operational platform that successfully automated workflow scheduling and secured multi-user permission tiers.
