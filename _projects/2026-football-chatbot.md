Responsive full-stack web application for football enthusiasts that enables dynamic exploration of live standings and match fixtures. The system leverages FastAPI to orchestrate server-side logic and features an optimized PostgreSQL database layer to process high-concurrency data streams fetched from external Football APIs. It integrates a custom Retrieval-Augmented Generation (RAG) pipeline to ingest real-time statistics into a Large Language Model (LLM), successfully eliminating knowledge cutoff boundaries and minimizing hallucination risks. Outcome: Built an enterprise-ready sports analytics chatbot capable of delivering contextually accurate, real-time insights for football league databases.
