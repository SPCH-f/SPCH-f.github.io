Comprehensive AI platform powered by Django and styled with Tailwind CSS that allows users to seamlessly generate digital artwork from textual descriptions. The platform processes raw text prompts into mathematical vector embeddings to establish semantic optimization boundaries before routing requests to generative models, ensuring maximum fidelity to user intent. It utilizes a structured MySQL backend to capture, index, and manage historical prompt metadata and image parameters for efficient user retrieval. Outcome: Built a highly accessible generative AI application with built-in query processing and persistent tracking infrastructure.
